## Project Overview
ReluxRent is a scalable Airbnb-style property rental platform that enables users to list properties, manage bookings, handle payments, communicate via messaging, and check real-time availability.  

---

## Features
- Full-stack rental platform built with Next.js, Express, Prisma, and MySQL for fast SSR and scalable data handling.
- Secure authentication with Firebase Auth and JWT, including OTP verification and role-based access.
- Property management dashboard with image uploads, availability & pricing control, and booking insights.
- Real-time chat, booking notifications, and a responsive UI using TailwindCSS, MUI, and Socket.io.
